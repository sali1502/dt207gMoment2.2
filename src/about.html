Om sidan

I laborationsuppgiften för moment 2 i kursen DT207G Backend-baserad webbutveckling har ett REST API som konsumeras med fetch API skapats. I den första backend-baserade delen, moment 2.1, har ett REST API byggts med Express och en mySQL-databas där CRUD (Create, Read, Update och Delete) har implementerats. I den andra frontend-baserade delen, moment 2.2, har en applikation skapats i en automatiserad utvecklingsmiljö med Parcel och SASS/SCSS. Webbplatsen är utformad som ett CV där det går att hämta och läsa ut arbetserfarenheter från webbtjänsten (GET), lägga till arbetserfarenheter (POST), uppdatera arbetserfarenheter (PUT) och radera arbetserfarenheter (DELETE). Jag har under arbetets gång fått en djupare insikt i processen kring att skapa ett REST API och att konsumera webbtjänsten med fetch API. Jag har särskilt fått övning i processen kring uppdatering av data via ett formulär.
